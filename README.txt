# PIRLS2026 — اختبار إلكتروني و لوحة تحليل

يحتوي هذا المجلد على:

- `index.html`: صفحة الاختبار.
- `dashboard.html`: لوحة تحليل النتائج (ترفع إليها ملفات CSV التي يتم تنزيلها من صفحة الاختبار).
- يستخدم الصور المرفقة: `t1.png`, `t2.png`, `t3.png`, `شعار القياس والتقويم التربوي.jpg`, `شعار رؤية عمان.png` بوجودها في المجلد الأعلى مباشرة.

## النشر على SharePoint (الموصى به داخل بيئة Microsoft 365)
1. افتح موقع SharePoint الخاص بالمديرية أو المدرسة.
2. ادخل إلى **Site Assets** أو مكتبة مستندات مخصّصة.
3. أنشئ مجلدًا باسم `PIRLS2026` وارفع محتويات هذا المجلد إليه مع الصور المرفقة في نفس المستوى (ضع الصور خارج مجلد `assets`).
4. انقر على `index.html` واختر **نسخ الرابط** ثم شاركه مع الطلبة بصلاحية العرض فقط.
5. لاختبار النتائج، افتح `dashboard.html` وارفع ملفات CSV التي حمّلها الطلبة بعد إنهاء الاختبار.

## ربط الإرسال التلقائي بالنتائج عبر Power Automate (اختياري)
- أنشئ تدفقًا جديدًا **When an HTTP request is received**.
- انسخ عنوان نقطة النهاية وضعه داخل المتغير `RESULTS_ENDPOINT` في `index.html`.
- اجعل التدفق يحفظ البيانات في **SharePoint List** أو **Excel** على OneDrive.
- أضف خطوة **Send an email (V2)** لإرسال إشعار إلى `waleedalshihi@moe.om` عند وصول نتيجة جديدة.

## الأمن والخصوصية
- لا يتم إرسال أي بيانات خارجياً إلا إذا تم ضبط `RESULTS_ENDPOINT`.
- بشكل افتراضي يمكن لكل طالب تنزيل ملف CSV الخاص بنتيجته وحفظه أو رفعه يدويًا.
