# pirls2026-exam.
اختبارات بيرلز (PIRLS) هي دراسة دولية هامة تقيس الفهم القرائي لطلاب الصف الرابع الأساسي بلغتهم الأم كل خمس سنوات، تشرف عليها منظمة IEA، وتساعد في مقارنة مهارات الطلاب عالميًا وتحديد مواطن القوة والضعف في تعليم القراءة، وتعتمد على نصوص سردية ومعلوماتية لتقييم القدرة على المتعة والتعلم من النصوص.
